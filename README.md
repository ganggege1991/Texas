### 已实现牌型
同花顺，四条 相互比较，各自比较

### 关键类简介

```Main``` 主函数入口，测试

```TexasHandler``` 处理接口，新的方式可以实现接口

```StraightFlushHandler``` 同花顺的具体处理

```FourBoomHandler``` 四条牌型具体处理


```HandlerManager``` 处理器的容器，添加新的处理方式，需要添加到容器里面
